# MeetRec v1.0 — снимок от 2026-07-05

- **MeetRec.dmg** — установщик (перетащить приложение в Applications).
- **MeetRec.app** — собранное приложение (macOS 15+, Apple Silicon, ad-hoc подпись).
- **source/** — полные исходники этой версии; пересборка: `./build.sh` внутри source.

Возможности версии:
- запись системного звука + микрофона в один .m4a (48 кГц, AAC);
- перетаскиваемое окно с кнопкой записи, таймером и списком последних записей;
- булавка «поверх всех окон», выбор папки записей, автоопределение Google Drive;
- значок и меню быстрых действий в строке меню, защита от повторного запуска.
