# MeetRec — запись встреч на Mac

Записывает **системный звук** (голоса собеседников в Zoom/Meet/Teams — в любом приложении) и **ваш микрофон** одновременно, сводит в один файл `.m4a`.

Есть два варианта: приложение с интерфейсом (основной) и утилита для терминала.

## Приложение MeetRec.app

Живёт в строке меню (значок волны). Уже установлено в `/Applications`.

- Клик по значку → окно с большой кнопкой записи.
- Во время записи в строке меню виден таймер; кнопка становится красной.
- После остановки дорожки сводятся и файл появляется в папке записей; последние записи открываются в один клик.
- Папку записей можно сменить кнопкой «Изменить» (по умолчанию `~/Documents/Записи встреч`).

### Разрешения (при первой записи)

macOS спросит разрешение на **запись экрана и системного звука** и на **микрофон** — выдайте их приложению MeetRec в «Системные настройки → Конфиденциальность и безопасность». Кнопка перехода в настройки появится прямо в окне приложения, если разрешения нет.

### Google Диск

Если установить [Google Drive для рабочего стола](https://www.google.com/drive/download/), MeetRec сам обнаружит его и предложит папку «Мой диск → Записи встреч» — записи будут автоматически синхронизироваться в облако.

## Установщик

`dist/MeetRec.dmg` — классический macOS-установщик: открыть, перетащить MeetRec в Applications. Удобно для переноса на другой Mac.

## Сборка

```sh
./build.sh   # собирает build/MeetRec.app и dist/MeetRec.dmg
```

Исходники приложения — в `app/Sources` (SwiftUI, меню-бар, macOS 15+). Иконка генерируется `app/make-icon.swift` и кэшируется в `app/AppIcon.icns`.

Дизайн — по скиллу [ui-ux-pro-max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) (установлен в `~/.claude/skills/ui-ux-pro-max`): палитра «Calm cyan + health green» (#0891B2 / #059669 / #DC2626), скругления 12 px, отступы 16 px, анимации 150–300 мс с учётом reduced motion.

## Утилита для терминала (альтернатива)

```sh
./meetrec                # записать в папку по умолчанию, остановка — Enter/Ctrl+C
./meetrec ~/Desktop      # в указанную папку
```

Или двойной клик по «Записать встречу.command». Пересборка: `swiftc -O -parse-as-library MeetRec.swift -o meetrec`.

## Технические детали

- ScreenCaptureKit: системный звук + микрофон, 48 кГц стерео, AAC 160 кбит/с на дорожку.
- Во время записи пишется временный `.mov` с двумя дорожками, при остановке они сводятся в один `.m4a`. Если сведение не удастся, «сырой» `.mov` сохраняется рядом — запись не теряется.
- Видео не записывается (для встреч обычно достаточно звука); при необходимости можно расширить.
