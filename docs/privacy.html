Политика конфиденциальности

Приложение MeetRec · обновлено 6 июля 2026
Коротко. MeetRec работает полностью на вашем устройстве. Приложение не собирает, не хранит на серверах и не передаёт третьим лицам ваши записи, транскрипты или любые персональные данные. У MeetRec нет учётных записей, аналитики и рекламы.

Какие данные обрабатываются

MeetRec создаёт и хранит только на вашем Mac: аудио- и видеозаписи встреч, текстовые расшифровки, итоги встреч и локальный индекс для поиска. Эти файлы находятся в выбранной вами папке (по умолчанию «Документы») и никуда не отправляются. Вы можете удалить их в любой момент.

Выход в интернет

Приложение обращается к сети только в двух случаях, и оба не связаны с передачей ваших данных:

  • Скачивание моделей. При первом использовании MeetRec загружает открытые модели распознавания речи и ИИ (с Hugging Face). Это одноразовая загрузка файлов моделей — никакие ваши записи при этом не отправляются.
  • Проверка обновлений моделей. Приложение периодически сверяется с публичным файлом-манифестом в репозитории проекта на GitHub, чтобы предложить более новую модель.

После загрузки моделей запись, расшифровка, разбор по спикерам, чат с ИИ и поиск по архиву работают полностью офлайн.

Google Календарь (необязательно)

Если вы сами подключите Google Календарь, MeetRec запрашивает доступ только на чтение (calendar.readonly), чтобы называть записи по встрече и напоминать о начале. Токены доступа хранятся в защищённой связке ключей macOS (Keychain) на вашем компьютере и используются исключительно для обращения к API Google Календаря напрямую с вашего устройства. MeetRec не пересылает данные вашего календаря на какие-либо сторонние серверы. Отключить доступ можно в один клик в самом приложении. Использование данных Google соответствует Google API Services User Data Policy, включая требования Limited Use.

Разрешения macOS

Для работы приложение просит системные разрешения на запись экрана и системного звука, микрофон и (по желанию) уведомления. Эти разрешения используются только для записи встреч по вашей команде.

Дети

Продукт не предназначен для детей младше 13 лет и не собирает их данные.

Изменения

Актуальная версия политики всегда доступна на этой странице. Существенные изменения будут отражены в дате обновления выше.
